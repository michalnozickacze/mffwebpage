Annotation

The lecture covers standard methods of data analysis, including modern trends of big data analysis using machine learning. Modelling over real data in the Python environment.


Syllabus:

  • Basics of linear regression, logistic regression, lasso regression, principles of hypotheses testing, likelihood ratio tests, stepwise algorithms
  • Basics of multidimensional statistics - principle component analysis, factor analysis, cluster analysis
  • Discrimination measures - Kolmogorov-Smirnov, Gini coefficient, Somer’s d
  • Back test principles, cross validation and bootstrapping
  • Regression trees, random forests
  • Gradient boosting
  • Bayes networks, neural networks
  • Linear optimization, Support vector machine


Exercise classes: Programming in Python, practical work with data.
